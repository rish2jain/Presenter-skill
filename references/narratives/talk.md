# Conference Talk Narrative

Use with `**Purpose:** talk` in outline front-matter.

## Recommended arc

| # | Layout | Content |
|---|--------|---------|
| 1 | `title` | Talk title + speaker |
| 2 | `full-image` or `bullet-list` | Hook — story or provocative claim |
| 3 | `section-divider` | Act 1 — context |
| 4–6 | varied | Core ideas (mix `bullet-list`, `stat-callout`, `two-column-split`) |
| 7 | `full-image` | Demo or hero visual |
| 8 | `timeline` | Journey / case study |
| 9 | `exec-summary` | 3 key takeaways |
| 10 | `closing` | Q&A + links |

Aim for **~1 slide per 2 minutes** of speaking time. Speaker notes are critical — write `- Notes:` for every slide.
