# Tutorial Narrative

Use with `**Purpose:** tutorial` in outline front-matter.

## Recommended arc

| # | Layout | Content |
|---|--------|---------|
| 1 | `title` | Topic + audience level |
| 2 | `bullet-list` | What you'll learn (outcomes) |
| 3 | `section-divider` | Concept 1 |
| 4 | `two-column-split` | Explanation + diagram |
| 5 | `full-image` | Screenshot / walkthrough |
| 6 | `bullet-list` | Step-by-step procedure |
| 7 | `comparison` | Before / after or wrong vs right |
| 8 | `timeline` | Workflow or pipeline |
| 9 | `exec-summary` | Summary + practice exercise |
| 10 | `closing` | Resources + contact |

Prefer light palette (`swiss-light`) for readability. Use `**Density:** comfortable` for teaching slides.
