# Internal Strategy Narrative

Use with `**Purpose:** strategy` in outline front-matter.

## Recommended arc

| # | Layout | Content |
|---|--------|---------|
| 1 | `title` | Initiative name + date |
| 2 | `exec-summary` | 3–4 strategic priorities |
| 3 | `section-divider` | "Situation" |
| 4 | `bullet-list` | Current state + constraints |
| 5 | `comparison` | Options considered |
| 6 | `two-column-split` | Recommended path + chart |
| 7 | `timeline` | Implementation roadmap |
| 8 | `stat-callout` | Expected impact (metrics) |
| 9 | `table` | Resource / budget summary |
| 10 | `closing` | Next steps + owners |

Set `**Takeaway:**` to the decision you want leadership to approve.
